### 2️⃣ Multithreading Dashboard

A real-time system that:
- Generates random numbers using multiple threads
- Displays them in:
  - Console
  - Colored terminal UI
  - GUI (Tkinter)
- Monitors CPU usage

#### 🔹 Features
- 6 concurrent threads
- Independent refresh rates
- Real-time visualization
- CPU usage monitoring
