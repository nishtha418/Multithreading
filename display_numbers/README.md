# Multithreading Dashboard

This project demonstrates multithreading in Python by generating random numbers concurrently and displaying them in different formats.

## Features
- 6 concurrent threads
- Independent refresh times
- Console display
- Colored terminal output
- GUI dashboard using Tkinter
- CPU usage monitoring using psutil

## Files
- code.py → basic console version
- code_colored.py → colored terminal UI
- code_gui.py → GUI version
- code_gui_cpu.py → GUI with CPU monitoring

## Concepts Used
- Python threading
- Shared data structures
- GUI updates with Tkinter
- Real-time system monitoring
